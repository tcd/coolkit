Useful ruby code with no dependencies.

## Installation

Add this line to your application's Gemfile:

```ruby
gem "coolkit"
```

And then execute:

```shell
bundle
```

Or install it yourself as:

```
gem install util
```

## Usage

```ruby
require "coolkit"
```
